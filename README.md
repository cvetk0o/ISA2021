Build Angular Project
  -Install NodeJs from NodeJs Official Page.
  -Open Terminal
  -Go to file project ..\ISA2021\frontIsa\
  -Run in terminal: npm install -g @angular/cli
  -Then: npm install
  -And: ng serve
  -Navigate to: http://localhost:4200/
